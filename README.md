# Weather App

**Мой первый pet-проект на чистом JavaScript**

---

## Описание проекта

**Weather App** — это простое и удобное приложение для просмотра текущей погоды и прогноза.
Позволяет искать города, видеть основные параметры погоды и сохранять избранные локации для быстрого доступа.

Проект создан в целях обучения, чтобы закрепить работу с **fetch API**, **асинхронностью**, **DOM-манипуляциями** и **обработкой пользовательского ввода**.

---

##  Функционал

* Поиск города
* Отображение текущей погоды
* Прогноз на ближайшие 3 интервала по 3 часа
* Иконки состояния погоды
* Добавление/удаление городов в избранное
* Повторный запрос кликом по избранному городу
* Обработка ошибок (пустой ввод, 404, проблемы с API)

---

## Стек технологий
* **HTML5** — разметка
* **CSS3** — стилизация, Grid + Flex
* **JavaScript** — вся логика
* **OpenWeather API** — получение данных о погоде

## Ключевые части логики

### Запросы к API

Используется `fetch()` + `async/await`
Получение:
* Текущей погоды (`weather`)
* Прогноза (`forecast`)
### Обновление интерфейса

Функции:
* `displayWeather()` — заполняет данные "сегодня"
* `displayForecast()` — рендерит 3 ближайших прогноза
###  Избранные города
* Добавляются динамически в DOM
* Клик по городу → повторный запрос
* Нажатие на `X` удаляет элемент
